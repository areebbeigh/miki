Focused reading mode

Toggle “Focus read” in the sidebar or popup to hide site chrome and show only the article. Width respects the Miki sidebar margin. Colors and typography apply while focus mode is on.

Site rules

One hostname per line. Lines starting with ! deny. Examples: wiki.example.com, *.internal.corp
